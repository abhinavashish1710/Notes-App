# 📝 Notes App

A lightweight command-line Notes App built with Python that allows users to create, view, and delete notes stored locally in a JSON file.

## 🚀 Features

- ➕ Add new notes
- 📋 View saved notes
- 🗑️ Delete notes
- 💾 JSON-based storage
- 🐍 Pure Python (No external libraries)

## 📂 Project Structure

```
Notes-App/
├── main.py
├── notes.json
├── requirements.txt
├── .gitignore
├── LICENSE
└── README.md
```

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/abhinavashish1710/Notes-App.git
```

Run the application:

```bash
python main.py
```
